# StartCraft II Imitation Learning
This project implemented DQN, Double DQN and Deep Q-Learning from Demonstration (DQfD).
## Requirements
* Python 3.6
* PyTorch 0.3
* PySC2
* Matplotlib
* gflags
* Numpy
